# Weather Dashboard

Real-time weather information for any city worldwide.

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)
- OpenWeatherMap API
- Netlify

## Live Demo
[https://weather-dashboard199.netlify.app](https://weather-dashboard199.netlify.app) *
(weather-dashboard199.netlify.app)*

## Features
- Search by city name
- Real-time weather data
- Temperature, conditions, humidity, wind speed
- Responsive design
- Error handling

## How to Run
1. Clone: `git clone https://github.com/akinlade-dev/weather-dashboard.git`
2. Get free API key from [openweathermap.org](https://openweathermap.org/api)
3. Replace `YOUR_API_KEY_HERE` in `app.js`
4. Open `index.html` in browser
